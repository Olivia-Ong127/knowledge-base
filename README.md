# 📚 数字员工·知识库

> 本知识库记录了一个"从零搭建服装行业数字员工决策框架"项目的完整思考过程。
>
> **创建时间：** 2026年6月
> **维护者：** Olivia Ong

---

## 这个知识库是什么

这不是代码仓库，也不是文档合集。这是**我的思考沉淀**——

在我们（我和Claude）的对话中，一次次讨论、纠偏、重构之后，留下的"为什么这么做"的记录。

## 内容结构

```
01-背景与画像/         ← 我、公司、联创人的背景信息
02-协作方式/           ← 我和Claude怎么配合工作
03-数字员工决策框架/   ← 核心产出：决策框架的各个模块
04-技术路径选择/       ← 技术选型的思考过程（用业务语言）
05-对话记录/           ← 重要对话的转折点和决策记录
06-附录/               ← 术语表、参考资源
```

## 怎么使用

- **新加入项目的人**：从 `01-背景与画像/` 开始读，了解来龙去脉
- **阶段性回顾的自己**：看 `05-对话记录/` 找到上次的决策点，再读对应模块
- **快速查找某个概念**：进 `06-附录/术语表.md`
