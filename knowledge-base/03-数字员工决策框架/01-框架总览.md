## 一句话定义

> 一个为服装行业商品调拨场景设计的AI决策助手框架。
> 它"知道什么" → "怎么思考" → "怎么和人配合"。

```

│    ┌──────┬──────┬──────┬──────┬──────┐                   │

│    └──┬───┴──┬───┴──┬───┴──┬───┴──┬───┘                   │
│       │      │      │      │      │                        │

│                  ↓                                         │

│    ┌──────┬──────┬──────┬──────┐                          │
│    │需求  │供给  │决策  │执行  │                          │
```

## 缺失维度清单

在对话中识别出框架目前缺少的关键维度：

1. **目标体系** — 调拨决策在优化什么？售罄率？毛利率？周转？
2. **技术路径选择** — 规则驱动还是AI驱动？实时还是批跑？
3. **决策分级/自治度** — AI能自己做什么？什么必须人批？
4. **异常兜底机制** — 数据缺失、规则冲突时怎么办？
5. **品类差异化** — 基本款、潮流款、配饰调拨逻辑不同

> 详见 `05-缺失维度分析.md`
